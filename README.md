# Arduino-Two-LEDs-Traffic-Light

## Contextualization
In this experiment, the LED_BUILTIN and two other LEDs will turn on and off sequentially using digital pin 13 and digital pin 12

## Components:
- 1 - Arduino Uno R3
- 1 - USB Cable 
- 5 - Jumpers(male-to-male)
- 1 - ProtoBoard
- 2 - LED
